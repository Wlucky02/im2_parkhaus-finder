Kurzbeschrieb
Wir kennen es alle. Mal wieder auf der Suche nach einem passenden Parkplatz. Wir haben die Lösung. Der Parkhausfinder Basel schlägt dir das passene Parkhaus vor und gibt dir die Anzahl freien Parkplätze live an, sodass du dir keine Sorgen machen musst ob das Parkhaus voll ist oder nicht. Mit der Navigation kannst du direkt deine Route planen und mit dem Stadtplan hast du alle Parkhäuser immer im Überblick. 

Learnings
Unsere Programmierkenntnise konnten wir stark verbessern, insbesonder JavaScript, welches neu dazukam. Mit Hilfe von GitHub haben wir gelernt, dass man den Code via GitHub teilen kann, sodass beide immer auf dem neusten Stand sind. Das Hauptthema war API. Wir lernten wie man externe Daten, in unserem Fall waren es die Anzahl freien Parkplätze, von einer externen Website anzapft und auf der eigenen Website einbindet. 

Schwierigkeiten
Wir hatten grosse schwierigkeiten mit dem GitHub und dem Code teilen, weshalb wir uns entschieden haben via Webex zusammenzuarbeiten, was hervorragend funktionert hat. Wir haben zusatzlich mit Swisstransfer gearbeitet, damit man von Zeit zu Zeit die Code einander zusenden konnte. Da wir uns die Unterseiten aufgeteilt haben, hatten wir kein Problem, dass wir den Code des anderen überschreiben. Was zusätzlich knifflig war, war die Website responsive zu machen, damit es auf dem Handy sauber lesbar ist. Vor allem bei den Kacheln hatten wir am anfang schwierigkeiten, wie wir die Mobile Anischt darfstellen möchten. Mit hilfe der Dozenten, sowie Studienkollegen und KI konnten wir schlussendlich ein Lösung finden.

Know Bugs
Das Hamburger Menü bei der Handyansicht ist nicht anschliessend an den Header. Der Grund ist das Logo. Wir haben aber nicht herausgefunden, wie wir das Menü über den Header nehmen können.
